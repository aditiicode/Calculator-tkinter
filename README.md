

This is a simple calculator application built using Python and Tkinter. It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- GUI-based calculator
- Supports +, -, *, / operations
- Simple and user-friendly interface
- Built using basic Python concepts

## Technologies Used
- Python
- Tkinter (GUI library)

## ▶️How to Run
1. Make sure Python is installed
2. Download or clone this repository
3. Run the file
